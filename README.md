# Orange Standard

A Bitcoin exchange built for simplicity and security. Buy, sell, and hold Bitcoin with US dollars.

## Why Orange Standard

- Real-time Bitcoin trading at market prices
- Institutional-grade custody and security
- Seamless USD funding via ACH and wire
- Two-factor authentication and identity verification
- Full-featured REST API for partners and developers

## Get started

Create an account, verify your identity, and start trading in minutes. Fund your account from any US bank.

## Trading

Buy and sell Bitcoin instantly at live market prices. Transparent fees shown before every order.

## Wallet

Deposit and withdraw both USD and Bitcoin. Generate deposit addresses, track your portfolio, and export your full transaction history.

## API

Integrate with Orange Standard through a REST API. Authentication, trading, wallets, and account management are all available programmatically. [Contact us](mailto:support@orangestandard.com) for API access.
